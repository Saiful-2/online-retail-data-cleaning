# 🛒 Online Retail Data Cleaning Project (Excel + Python)

## 📊 Project Overview

This project demonstrates a complete data cleaning workflow using Microsoft Excel and Python (Pandas) on a real-world e-commerce dataset.

---

## 🔧 Tools Used

* Microsoft Excel
* Python (Pandas)
* Jupyter Notebook

---

## 🔹 Excel Data Cleaning

* Removed canceled transactions (InvoiceNo starting with 'C')
* Handled missing CustomerID values
* Removed negative quantities and invalid prices
* Cleaned text fields (Description column)
* Created a new feature: **TotalPrice (Quantity × UnitPrice)**

---

## 🔹 Python Data Processing

* Loaded cleaned dataset using Pandas
* Checked missing values and duplicates
* Converted data types (InvoiceDate)
* Created new features: **Month and Year**
* Validated TotalPrice calculations

---
## 📊 Key Features

- End-to-end data cleaning workflow (Excel + Python)
- Real-world messy dataset handling
- Feature engineering for business insights
- Data validation and consistency checks
